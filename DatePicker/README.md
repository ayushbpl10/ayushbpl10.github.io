# Angular 2 Datepicker

It's an Angular 2 project for the datepicker.

# Prerequisites
You must have node.js and npm installed on your machine.

# Install

Clone the project in directory.

Now , Open Cmd on Window.
Navigate to the specific location of the directory "Angular2-datepicker " .
Inside the " Angular2-datepicker " directory.

Run command

```
npm install
```
Then Run Command

```
npm start
```

A browser window will automatically open. 

# Usage

* You can select a single date by clicking it.
* You can select multiple random dates by pressing CTRL key while selecting dates.
* You can select range of dates by pressing SHIFT key while selecting start date and end date.
